# PRD: 세무법인 아람 상담 서비스

## 목표
세무법인 아람의 공식 홈페이지. 잠재 고객이 온라인으로 세무 상담을 신청하고 진행 현황을 조회할 수 있는 서비스.

## 사용자
- **상담 신청자**: 세무 관련 문의가 있는 개인 또는 법인
- **세무법인 관리자**: Django Admin을 통해 상담 내역 확인 및 답변 작성

## 핵심 기능 (MVP)

### 1. 상담 신청
- 상담 유형 선택: 법인세 / 소득세 / 부가가치세 / 종합소득세 / 기타
- 신청자 정보 입력: 이름, 이메일, 연락처
- 조회용 비밀번호 설정 (상담 조회 시 본인 확인용)
- 상담 내용 작성 (선택)
- 첨부파일 업로드 (선택, 복수 파일)

### 2. 이메일 발송
- 상담 신청 완료 시: 관리자에게 새 상담 접수 알림 발송
- 관리자 답변 작성 시: 신청자에게 답변 완료 알림 발송

### 3. 상담 조회
- 이메일 + 비밀번호로 본인 상담 내역 확인
- 상담 유형, 신청일, 진행 상태(접수대기/진행중/완료) 표시
- 관리자 답변 내용 확인
- 첨부파일 다운로드

## MVP 제외 사항
- 회원 가입 / 로그인 (이메일+비밀번호 직접 조회로 대체)
- 실시간 채팅 상담
- 상담 일정 예약 캘린더
- 관리자 전용 웹 대시보드 (Django Admin으로 대체)
- 컬럼/블로그 페이지 (향후 추가 예정)
- 다국어 지원
- 이메일 인증

## 향후 기능 (Post-MVP)
- **컬럼/블로그 페이지**: 세무법인 대표/직원이 작성하는 세무 정보 콘텐츠. 관리자가 글을 게시하면 홈페이지에 노출.
- 관리자 커스텀 대시보드
- 이메일 인증 기반 조회
- 상담 현황 SMS 알림
